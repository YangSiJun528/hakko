아직 테스트 단계입니다.

CHZZK 스트리머 SOOP 스트리머

* 본 서비스는 각 플랫폼의 공식 서비스가 아닙니다.

프로젝트 소개

hakko는 시청자 수가 적은 스트리머들을 쉽게 발견할 수 있도록 도와주는 서비스입니다.

주요 기능

  • 0명의 시청자를 보유한 스트리머 무작위 추천
  • CHZZK, SOOP 플랫폼 지원
  • 실시간 방송 시청

개발자 소개

안녕하세요! hakko을 개발한 입니다.

작은 스트리머들이 더 많은 시청자와 만날 수 있도록, 시청자들은 새로운 스트리머를 쉽게 발견할 수 있도록 이 프로젝트를 시작하게 되었습니다.

이 프로젝트는 nobody.live에서 영감을 받았습니다.
